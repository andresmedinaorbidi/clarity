## INSTRUCTIONS:

1. **Be Specific:** Use actual details from the research data. Don't be generic.
2. **Sound Expert:** Write like a senior consultant presenting to a client.
3. **Fill Every Section:** Even if you need to make educated inferences, complete all sections.
4. **Use Research Data:** Reference specific insights from the research_data when available.
5. **Match the Brand:** If colors/style are provided, incorporate them into Visual Direction.

{instruction}

**IMPORTANT:** 
- Output ONLY the Markdown content. Do not wrap it in code blocks. 
- Do not add any preamble or explanation before or after the brief.
- NEVER include JSON structures, code blocks, or raw data in your output.
